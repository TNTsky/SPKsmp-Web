SERVER FAQ

伺服器基本信息

目前伺服器為JAVA服，透過Geyser模組使 BE 玩家也可加入，因此遊戲機制以JAVA版為主。

若沒有JAVA版帳號，也可透過 BE 加入伺服器。但若您同時具有 JE 與 BE ，請注意兩者的存檔是分開的。

是的，可以透過連結雙版本帳號使不同版本(裝置)遊玩相同存檔，詳細方法請至 DC 群的公告釘選訊息查看。

模組與插件

我們有安裝一些利於遊玩生電，如地毯(Carpet)等不改變原有玩法的模組。

為了使 Bedrock 版本也可正常遊玩，玩家不需安裝任何模組即可進入伺服器，但 Java 版本可自行安裝客戶端模組以獲得更好的遊戲體驗。

我們有提供推薦的模組整合包可供使用，但請勿使用任何違反伺服器規則的外掛。

贊助與支持

本伺服器為非營利性質，目前不提供任何課金服務，所有伺服器相關費用皆由管理員們自行承擔。如果您喜歡本伺服器並願意贊助，我們將不勝感激。

贊助將不會在伺服器內獲得任何特權，但若您有任何需求或是希望事項，如自訂身分組、更換伺服器頭像、MOTD等，我們將盡力配合。

活動與管理

我們不定期舉辦各種活動，但因人手有限，活動頻率較低。

請通報管理員，我們將會進行調查並採取相應措施。

您隨時可於聊天頻道中提出意見，或直接聯繫管理員皆可。

伺服器介紹

歡迎來到 Sparkle 花火伺服器！你知道我們為什麼叫做 Sparkle 嗎？這是一個有趣的故事！

有一次，我們的紅石機器壞掉了，結果爆炸了，炸出了一片絢麗的花火，於是我們決定以這個奇妙的事件命名我們的伺服器——Sparkle！

在 Sparkle Server，我們提供充滿樂趣和創意的 Minecraft 伺服器體驗。不論你是 Java 版還是基岩版（BE）的玩家，都可以在這裡找到屬於自己的樂園。

我們目前有兩個主要的伺服器：

  • 生存服：在這裡，你可以探索廣闊的世界，與其他玩家合作建造宏偉的建築，或者獨自冒險征服各種挑戰。
  • 創造服：釋放你的創意！在這個伺服器中，你可以自由地建造任何你能想像到的東西，不受資源限制。

無論你是喜歡生存模式的挑戰，還是享受創造模式的自由，Sparkle Server 都能滿足你的需求。加入我們，開始你的 Minecraft 冒險吧！
